Overview

Default XFCE is fast, but visually dated. This guide documents my specific setup to force XFCE into a modern, consistent macOS Big Sur Dark aesthetic using the WhiteSur suite.

1. Dependencies

The install scripts require these packages. Do not skip this.

sudo apt update && sudo apt upgrade -y sudo apt install git sassc plank xfce4-appmenu-plugin rofi imagemagick -y

2. Typography

San Francisco (SF Pro) fonts are mandatory for the correct look.

mkdir -p ~/.local/share/fonts git clone https://github.com/sahibjotsaggu/San-Francisco-Pro-Fonts.git cp San-Francisco-Pro-Fonts/*.otf ~/.local/share/fonts/ fc-cache -f -v

Navigate to Settings > Appearance > Fonts and apply SF Pro Display globally.

3. Main Assets

We are using the WhiteSur suite. I use the 'Solid' variant to avoid compositing glitches on XFCE.

GTK Theme

git clone https://github.com/vinceliuice/WhiteSur-gtk-theme.git --depth=1 cd WhiteSur-gtk-theme ./install.sh -c Dark -t all -N Glassy -s

Icon Theme

cd .. git clone https://github.com/vinceliuice/WhiteSur-icon-theme.git --depth=1 cd WhiteSur-icon-theme ./install.sh -b

Cursor Theme

cd .. git clone https://github.com/vinceliuice/McMojave-cursors.git --depth=1 cd McMojave-cursors ./install.sh

4. System Configuration

Apply the assets in Settings Manager:

  • Appearance > Style: WhiteSur-Dark-solid
  • Appearance > Icons: WhiteSur-dark
  • Window Manager > Style: WhiteSur-Dark-solid
  • Mouse > Theme: McMojave-cursors

5. Dock (Plank)

Disable the default XFCE bottom panel before running Plank.

cd WhiteSur-gtk-theme/src/other/plank cp -r * ~/.local/share/plank/themes/

Run plank --preferences. Set theme to WhiteSur-Dark and enable Icon Zoom.

6. Spotlight (Rofi)

Replaces the default application menu with a spotlight-like search.

mkdir -p ~/.config/rofi git clone https://github.com/adi1090x/rofi.git cd rofi && ./setup.sh

Bind Super + Space to your preferred Rofi launch command.

7. Boot & Lockscreen

Ensure visual consistency from boot to desktop.

Plymouth

Warning: Incorrect Plymouth config can cause black screens. Proceed with caution.

git clone https://github.com/vinceliuice/WhiteSur-plymouth.git sudo ./WhiteSur-plymouth/install.sh

LightDM

Use GTK Greeter Settings to safely theme the lockscreen.

sudo apt install lightdm-gtk-greeter-settings sudo lightdm-gtk-greeter-settings

Set Theme/Icons to WhiteSur-Dark inside the settings app.

Setup Complete

Reboot your system to apply all changes properly.
