## âš¡ About The Project

![Preview Image](Preview.png)

Default XFCE is fast and lightweight, but visually outdated. This project serves as a comprehensive guide to force XFCE into a modern, consistent macOS Big Sur aesthetic using the **WhiteSur** suite by Vinceliuice.

The documentation covers every aspect of the "ricng" process, focusing specifically on the **Dark Mode** variant to ensure visual consistency across GTK apps, the shell, and the dock. It acts as a central hub for all necessary terminal commands, dependencies, and configuration steps.

## ğŸ’¡ Usage

This repository is designed to be viewed as a static website. It guides you through the following phases:

1.  **Dependencies:** Installing `git`, `sassc`, `plank`, etc.
2.  **Typography:** Setting up San Francisco (SF Pro) fonts.
3.  **Assets:** Installing WhiteSur GTK, Icons, and Cursors.
4.  **Dock:** Configuring Plank with macOS theme.
5.  **Spotlight:** Setting up Rofi launcher.
6.  **System:** Configuring Plymouth boot logo and LightDM lockscreen.

> **Note:** For the full guide and copy-pasteable commands, please visit the [Project Website](https://hndrapratamaa.github.io/XFCE-BigSur).
